# LifeOrbit

目標・タスク・スケジュールの統合管理Webアプリケーション。

## 🚀 特徴
- **統合管理**: 目標（Goals）、タスク（Tasks）、スケジュールを1つのダッシュボードでシームレスに確認・管理。
- **カレンダー連携**: グループウェア等のカレンダーからエクスポートした `.ics` ファイルをドラッグ＆ドロップするだけでスケジュールをインポート。
- **端末間同期**: 面倒なアカウント登録をせず、自分専用の「合言葉（同期コード）」を入力するだけでスマホやPCなど複数端末間でリアルタイムに同期・共有。

## 🛠️ 技術スタック
- フロントエンド: HTML5 / Vanilla CSS3 / Vanilla JavaScript
- データベース/同期: Firebase Firestore (デフォルト埋め込み)
